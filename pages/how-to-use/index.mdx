# Pakegai のつかいかた

ここからは、Pakegai の使い方をご紹介します！

このアプリには大きく3つの機能があります。「おすすめ・ホーム・本棚」の3つです。
画面下部にあるタブバーを押すことでそれぞれの画面へ遷移できます。

<Image src="/screens/tabbar.webp" width={400} height={300} alt="タブバー"/>

- おすすめ画面のつかいかたは[コチラ](/how-to-use/recommend)をクリック
- ホーム画面のつかいかたは[コチラ](#ホーム画面のつかいかた)をクリック
- 本棚画面のつかいかたは[コチラ](/how-to-use/bookshelf)をクリック

## ホーム画面のつかいかた

ホーム画面は Pakegai のメイン画面です🌟


ホーム画面では、世界中にある本の一覧を見ることができます。
画面上部の検索バーから、本のタイトルや著者名を入力することで、
その本を検索することもできます。

<Image src="/screens/home/index.webp" width={400} height={1200} alt="ホーム画面"/>

また、本のタイトルの横にある「･･･」を押すことで、
その本を本棚に追加することができます。

<Image src="/screens/home/add-bookshelf.webp" width={400} height={1200} alt="ホーム画面での本棚の追加"/>

あなたの人生に役立つ本を、ぜひ見つけてください！📚
